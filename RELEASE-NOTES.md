# Moprius Font Family v1.0.0

First combined public release of the complete Moprius type system.

## Included

- Moprius Sans 1.437: Regular, Italic, Bold, Bold Italic.
- Moprius Mono 1.437: Regular, Italic, Bold, Bold Italic.
- Moprius Serif 1.000: Regular, Italic, Bold, Bold Italic.
- Desktop TTF/OTF files and WOFF2 webfonts.
- Combined CSS, editable TTX sources, specimens and audit tools.

## Intended use

- Sans: desktop UI and general text.
- Mono: programming, terminal and tabular text.
- Serif: books, articles, essays and long-form reading.
